
# HerHaven 🌸

**HerHaven** is a web-based wellness platform for women — offering everything from menstrual tracking and pregnancy support to mental health resources, all in one beautiful pastel-pink themed space.

### 🛠️ Built With:
- HTML, CSS, JavaScript (Frontend)
- MySQL, PHP / Node.js (Backend)
- C++ modules for data processing (optional)
- Canva / AI design tools for UI/UX

### 💡 Features Planned:
- Period & Ovulation Tracker
- Pregnancy Stage Insights
- Mental Wellness Journal
- Secure User Login & Data Storage
- Support Resources & Helpline Finder
